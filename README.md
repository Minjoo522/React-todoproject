## 주요기능
1. 할 일 입력
2. 할 일 삭제
3. 완료한 일 체크
4. 전체 / 아직 하지 않은 일 / 완료한 일 필터링해서 보기

6. 로컬스토리지에 할 일들과 테마 저장하여 기억
